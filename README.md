# lynx-soft-test

Страница фильтрации данных пациентов на Vue 3, Vuetify 3 и TypeScript.
Сделана адаптивная вёрстка для десктопа и мобилок.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Линтинг и форматирование

```bash
npm run lint
npm run format
```
